# Anticuchería Melchorita V1

Este proyecto es la página web de la Anticuchería Melchorita.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

- `index.html`: Archivo principal de la página web.
- `assets/`: Carpeta que contiene todos los recursos estáticos.
    - `css/`: Hojas de estilo (CSS).
    - `js/`: Scripts de JavaScript.
    - `img/`: Imágenes del proyecto.

## Cómo ver el proyecto

Simplemente abre el archivo `index.html` en tu navegador web de preferencia.

## Tecnologías

- HTML5
- Tailwind CSS (vía CDN)
- JavaScript
